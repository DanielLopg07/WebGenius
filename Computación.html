LibreOffice

LibreOffice es una suite ofimática de código abierto y gratuita, desarrollada por The Document Foundation. Es una alternativa a otras suites ofimáticas como Microsoft Office. LibreOffice incluye las siguientes aplicaciones principales:

  • Writer: un procesador de textos similar a Microsoft Word.
  • Calc: una hoja de cálculo similar a Microsoft Excel.
  • Impress: una herramienta para crear presentaciones similar a Microsoft PowerPoint.
  • Draw: una herramienta de dibujo y diagramas.
  • Base: una base de datos similar a Microsoft Access.
  • Math: un editor de fórmulas matemáticas.


OpenOffice

Apache OpenOffice (anteriormente conocido como OpenOffice.org) es otra suite ofimática de código abierto y gratuita. Originalmente fue desarrollado por Sun Microsystems y ahora es mantenido por la Fundación Apache. Las aplicaciones principales de OpenOffice son:

  • Writer: procesador de textos.
  • Calc: hoja de cálculo.
  • Impress: presentaciones.
  • Draw: gráficos y diagramas.
  • Base: base de datos.
  • Math: editor de fórmulas matemáticas.


Herramientas de Google

Las herramientas de Google son una serie de aplicaciones en línea integradas en Google Drive, que permiten la creación, edición y almacenamiento de documentos en la nube. Estas herramientas incluyen:

  • Google Docs: un procesador de textos en línea similar a Microsoft Word y LibreOffice Writer.
  • Google Sheets: una hoja de cálculo en línea similar a Microsoft Excel y LibreOffice Calc.
  • Google Slides: una herramienta para crear presentaciones en línea similar a Microsoft PowerPoint y LibreOffice Impress.
  • Google Forms: una herramienta para crear formularios y encuestas en línea.
  • Google Drawings: una aplicación para crear dibujos y diagramas.
  • Google Keep: una herramienta de notas y listas.
  • Google Sites: una aplicación para crear sitios web sencillos.

Las herramientas de Google se destacan por su capacidad de colaboración en tiempo real, donde múltiples usuarios pueden trabajar simultáneamente en el mismo documento, hoja de cálculo o presentación. También se integran perfectamente con otros servicios de Google, como Google Calendar y Gmail.
